# ChatApp

## Technologies

Client: React

Server: Node

DataBase: MongoDB

## Getting Started

### Installing
```
git clone https://github.com/dorhazan99/TalkBackApp
```
### Install dependencies
```
npm install
```
```
cd .\frontend\
npm install
```
## Start the server
```
cd .\backend\
npm start
````
## Start the client
```
cd .\frontend\
npm start
````
